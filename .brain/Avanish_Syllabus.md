## ☕ Track 1: Data Structures & Algorithms (TUF+ Java)
*Language: Java | Reference: Striver's TUF Plus and GATE PYQs*
*Overlap: 100% (Core of both PSU CBTs and SDE interviews)*

- **Beginner Problems**: Language Basics, Basic Maths, Basic Recursion, Basic Hashing.
- **Sorting Algorithms**: Bubble, Selection, Insertion, Merge, Quick, Counting Sort.
- **Arrays**: 1D/2D arrays, sliding window, two-pointer, prefix sum, hashing.
- **Linked Lists**: Singly, Doubly, Circular, Floyd's Cycle detection, reversals.
- **Stacks & Queues**: Implementations, Monotonic Stacks, expression evaluation.
- **Trees**: Binary Trees, BSTs, Traversals, Height, Diameter, Balanced Trees.
- **Graphs**: BFS, DFS, Dijkstra, Bellman-Ford, Kruskal, Prim, Floyd-Warshall.
- **Dynamic Programming**: 1D, 2D, Grid, Stocks, Strings, MCM DP.

## 🌐 Track 2: Web Development & DevOps (Harkirat Bootcamp)
*Language: JavaScript / TypeScript / Node.js*
*Overlap: 10% (SDE-specific, essential for job switch portfolio)*

- **Frontend**: HTML, CSS, React.js, Next.js, Tailwind CSS, State Management.
- **Backend**: Node.js, Express, REST APIs, RPC, GraphQL, WebSockets.
- **Databases**: PostgreSQL, MongoDB, Prisma ORM, Redis (Caching).
- **DevOps**: Docker, Kubernetes, CI/CD, AWS/Azure deployments.

## 🤖 Track 3: AIML (Harkirat Cohort)
*Language: Python | Reference: Harkirat's AIML Bootcamp*
*Overlap: 20% (SDE-specific, essential for AI Engineer switch)*

- **Applications**: Retrieval-Augmented Generation (RAG), LangGraph, Agentic AI.

## 🏗️ Track 5: Low Level Design (LLD) & OOPS
*Overlap: 100% (Core of SDE-2 interviews and PSU interview/CBT questions)*

- **OOPS**: Classes, Objects, Inheritance, Polymorphism, Encapsulation, Abstraction.
- **SOLID**: Single Responsibility, Open/Closed, Liskov Substitution, Interface Segregation, Dependency Inversion.
- **Design Patterns**: Creational (Singleton, Factory), Structural (Adapter, Decorator), Behavioral (Observer, Strategy).

---

## 🗄️ Track 6: DBMS & SQL
*Overlap: 100% (Core of both PSU CBTs and SDE interviews)*

- **Theoretical DBMS**: ER Diagrams, Relational Model, Keys, Integrity Constraints.
- **Normalization**: Functional Dependencies, 1NF, 2NF, 3NF, BCNF.
- **SQL**: DDL/DML, Joins, Subqueries, Aggregate Functions, HAVING, Triggers.
- **Transactions**: ACID properties, Concurrency Control, 2-Phase Locking (2PL), Deadlocks.

---

## 💻 Track 7: Operating Systems (OS)
*Overlap: 100% (Core of both PSU CBTs and SDE interviews)*

- **Processes**: Process States, PCB, Context Switching, Threads (ULT vs KLT).
- **CPU Scheduling**: FCFS, SJF, Round Robin, Priority Scheduling.
- **Synchronization**: Semaphores, Mutex, Classic Problems (Producer-Consumer, Dining Philosophers).
- **Memory Management**: Paging, Segmentation, Virtual Memory, Page Replacement Algorithms.
- **Disk & Files**: Disk Scheduling, File Systems, Directories.

---

## 🔌 Track 8: Computer Networks (CN)
*Overlap: 100% (Core of both PSU CBTs and SDE interviews)*

- **Models**: OSI Reference Model, TCP/IP Suite.
- **Physical & Data Link**: Framing, Error Control, Flow Control (Sliding Window).
- **Network Layer**: IP Addressing (IPv4 vs IPv6), Subnetting, CIDR, Routing (Dijkstra).
- **Transport Layer**: TCP vs UDP, Congestion Control, Flow Control.
- **Application Layer**: HTTP, DNS, DHCP, SMTP, SSL/TLS.
